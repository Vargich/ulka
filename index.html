Дино костюм

Дино костюм для малыша станет любимой вещью в гардеробе 😍. Большой выбор расцветок, любой размер.

Спортивный костюм с накладными карманами

Спортивный костюм из футера трехнитки с начёсом. С контрастными накладными карманами. Расцветки в ассортименте.

Спортивный костюм с толстовкой с контрастным капюшоном и нашивкой

Спортивный костюм из футера трехнитки с начёсом. Капюшон сшит из принтованного контрастного футера двухнитки. Нашивка выполнена методом сублимационной печати. Можно напечатать любое изображение/фото/надпись по вашим пожеланиям.

Лонгслив для мальчика с комбинированными рукавами

Лонгслив из однотонной плотной кулирки в сочетании с ярким принтом на рукавах. Ткани в ассортименте.

Бомбер утеплённый

Бомбер из футера трехнитки с начёсом. Расцветки в ассортименте. Подберём по вашим предпочтениям.

Костюм кофта на кнопках брюки со стрелками и имитацией ширинки

Яркий утеплённый костюм из футера трехнитки с начёсом. Стрелки и имитация ширинки делают брюки более универсальными, что даёт возможность использовать не только как спортивные. Расцветки в ассортименте. Можно комбинировать цвета.

Детский спортивный костюм с комбинированными рукавами

Костюм выполнен из футера трехнитки с начёсом, вставка на рукавах из футера. Расцветки в ассортименте. Костюм можно дополнить нашивкой.
